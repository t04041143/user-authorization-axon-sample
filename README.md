# User Authorization Axon Sample
用于学习Axon Framework的样例，使用前后端分离开发模式，前端用reactjs + dva，服务端使用spring boot + axon。

## 安装

### 依赖的SDK
+ NodeJS 8.X.X以上
+ JDK 1.8
+ Maven 3.X.X

### 数据库
MongoDB 4.X.X
